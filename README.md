# mock-data
It generates a `mock-data.json` file with 100 objects.

Object example:
```
{
  "lastName": "Swift",
  "firstName": "Carmine",
  "nino": "6925",
  "address": {
    "street": "7022 Bryon Shoals",
    "city": "Tylerview",
    "county": "Avon"
  },
  "phone": "256-764-0829"
}
```

## Usage

__Install the dependencies:__

`npm install`

__Create file:__

```npm start```
